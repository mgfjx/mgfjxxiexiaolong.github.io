Shell 使用小技巧

2017-01-18
Shell
shell

1、sed 's/}/text}/g' file:

在文件file文件中的所有}之前添加text文本.
例如:NSLog(@"%@",NSStringFromSelector(_cmd));打印当前方法名

waring:如果遇到sed: RE error: illegal byte sequence, 这是因为在识别含有多字节编码字符时遇到了解析冲突问题, 请在终端执行export LC_COLLATE='C'和export LC_CTYPE='C'

2、将文件夹下的所有文件或文件夹按字母分类

用法：bash script.sh /directory/path

3、find技巧

1.批量查找并拷贝到指定目录：
find path -name "*.gif"| xargs -I {} cp {} targetPath

2.查找指定路径下所有文件并忽略隐藏文件及隐藏文件夹下的所有文件：
find <dir> -path "*/.*" -prune -o -print

3.查找指定路径下指定文件并忽略隐藏文件及隐藏文件夹下的所有文件：
find $PWD -path '*/.*' -prune -o \( -name '*.py' -o -name "*.sh" \) -type f -print
